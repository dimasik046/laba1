# laba1
Лабораторная работа №1: Работа с Git и GitHub

Описание проекта: Данный проект создан в рамках выполнения лабораторной работы №1. Цель работы: изучение платформы GitHub и практическое использование Git для хранения, обновления и распространения исходного кода проекта.

Задание по варианту: Вариант 1 — Геометрические фигуры.

Реализованы функции для вычисления параметров следующих фигур (распределены между участниками группы): Треугольник – периметр, площадь по формуле Герона, проверка
на равнобедренность; Трапеция – периметр, площадь, длина средней линии.

Выполнил: Счастливых Дмитрий — Треугольник – периметр, площадь по формуле Герона, проверка
на равнобедренность;
